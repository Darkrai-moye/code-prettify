100% automatic code highlighting using the Prettify library. No shortcodes, no bullshit.


== Description ==

Uses the [Google Code Prettify library](https://code.google.com/p/google-code-prettify/) 
that is minified and properly enqueued. Plugin applies code highlighting 
automatically to all `<pre>` tags on the page.

== Installation ==

Simplify upload the plugin and active it. It works out of the box and doesn't use an options panel.
